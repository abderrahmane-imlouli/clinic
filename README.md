# Clinic Appointment System

A Django-based backend application for managing clinic appointments.  
This project demonstrates my skills in backend development, database management, and web application logic.

## Features

- Patient registration and management
- Doctor scheduling and availability
- Appointment booking and tracking
- Admin dashboard for managing doctors and appointments
- RESTful API endpoints for frontend integration
- Secure authentication for users and admins

## Technologies Used

- Python
- Django
- SQLite (or PostgreSQL if upgraded)
- Django REST Framework (for API endpoints)

## Purpose of the Project

This project helped me:

- Learn backend development with Django
- Understand models, views, and templates
- Build RESTful APIs for web applications
- Practice authentication and role-based access
